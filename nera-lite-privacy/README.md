# Nera Lite Privacy Policy

Lightweight static privacy policy page for `Nera Lite`, ready for GitHub and Cloudflare Pages deployment.

## Files

- `index.html`: main privacy policy page
- `styles.css`: responsive Apple-inspired styling

## Deploy

1. Push the `nera-lite-privacy` folder into your GitHub repository.
2. In Cloudflare Pages, connect the repository.
3. Set the build command to empty.
4. Set the output directory to `nera-lite-privacy`.
5. Add your custom domain in Cloudflare Pages after the first deployment.
